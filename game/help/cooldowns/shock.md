# Cooldown: Shock
Usage: `shock [target]`
Cooldown: 6s
Cost: 5% `MP`
Type: `Spell`
Saving Throw: `Will`
Cast Time: 3s

Electricity envelops the target applying a 30 second shocked (`debuff-shock`)
debuff that deals 20-30 `potency` damage (based on `evocation` proficiency) to
the target each round.

See also: `evocation` (skill)
