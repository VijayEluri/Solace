# Cooldown: Aetherflow
Usage: `aetherflow`
Cooldown: 300s
Special: Gnomish race only

The aetherflow command is a special racial cooldown available only to gnomes.
When executed the player will immediately heal 50% of their max `MP`.

See also: `gnome` (race)
